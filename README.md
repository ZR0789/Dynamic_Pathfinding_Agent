# Dynamic_Pathfinding_Agent
Dynamic Agent is a grid-based pathfinding visualization tool built using Python and Pygame.
It demonstrates how informed search algorithms behave in both static and dynamic environments, allowing users to observe real-time exploration, frontier expansion, replanning, and final path construction.

This project was developed to bridge the gap between theoretical AI search concepts and practical, visual intuition — especially for understanding how algorithms react when the environment changes during execution.

 Overview

Unlike basic pathfinding visualizers, Dynamic Agent introduces:
 Support for informed search (A* and Greedy Best-First Search)
 Multiple heuristics (Manhattan & Euclidean)
 Dynamic obstacle spawning during execution
 Real-time replanning
 Fully interactive UI with performance metrics

It is designed for students studying Artificial Intelligence, Search Algorithms, or Heuristic Optimization.
 Features
 Interactive Grid Environment
Adjustable grid size (rows & columns)
Adjustable obstacle density
Click-to-edit cells:
Walls
Start node
Goal node
Optional coordinate display

 Implemented Algorithms
The system visualizes the following informed search algorithms:
A* (A-Star Search)
Greedy Best-First Search (GBFS)

Both algorithms operate in the same environment for easy comparison.
